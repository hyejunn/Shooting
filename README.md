# 자바 Swing을 활용한 슈팅게임

AWT, Swing을 사용한 자바 GUI

## Install
[Java version 19](https://jdk.java.net/java-se-ri/11-MR2) 설치

윈도우 설정 - 시스템 - 정보 - 고급 시스템 설정 - 환경변수 - 시스템 변수 - 새로 만들기

- 변수 이름 : JAVA_HOME
- 변수 값 : (자바 설치 경로) ex) C:\Program Files\Java\jdk-11\bin
