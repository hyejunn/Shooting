# 자바 Swing을 활용한 슈팅게임

## Install
[Java version 19](https://www.oracle.com/kr/java/technologies/downloads/#jdk19-windows) 설치

윈도우 설정 - 시스템 - 정보 - 고급 시스템 설정 - 환경변수 - 시스템 변수 - 새로 만들기

- 변수 이름 : JAVA_HOME
- 변수 값 : (자바 설치 경로) ex) C:\Program Files\Java\jdk-19\bin
